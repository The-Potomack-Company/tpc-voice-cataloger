**B. Make vite.config.ts conditional for basicSsl (DEPLOY-01 prerequisite):**

Change `vite.config.ts` from static `defineConfig({...})` to function-based `defineConfig(({ command }) => ({...}))`. The `basicSsl()` plugin should only load when `command === 'serve'` (dev server). During `vite build` (CI and Vercel), it must NOT load.

Current (line 8-13):
```typescript
export default defineConfig({
  server: { host: true },
  plugins: [
    basicSsl(),
```

Change to:
```typescript
export default defineConfig(({ command }) => ({
  server: { host: true },
  plugins: [
    command === 'serve' ? basicSsl() : null,
    react(),
    tailwindcss(),
    VitePWA({ /* ...existing config unchanged... */ }),
  ].filter(Boolean),
  test: {
    globals: true,
    environment: "jsdom",
    setupFiles: ["src/tests/setup.ts"],
  },
}));
```
