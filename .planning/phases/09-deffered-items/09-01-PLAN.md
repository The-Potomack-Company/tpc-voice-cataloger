2. Add `unarchiveSession` function (removes archivedAt using modify + delete pattern, same as restoreSession removes deletedAt):
```typescript
export async function unarchiveSession(id: number): Promise<void> {
  await db.sessions.where("id").equals(id).modify((session) => {
    delete session.archivedAt;
    session.status = "active";
    session.updatedAt = new Date();
  });
}
```
Note: Un-archiving also sets status to "active" per CONTEXT.md decision ("Un-archiving moves session back to active list and unlocks it for editing").
