    1. Uses `createImageBitmap(file)` to load image (handles EXIF orientation automatically)

    3. Uses `OffscreenCanvas` if available, otherwise falls back to regular `<canvas>` element
    4. Draws image and converts to JPEG blob at 0.85 quality

    Create `RECEIPT_PATTERN = /^\d{5}-\d+$/` and `isValidReceiptNumber(value: string): boolean` that trims whitespace and tests against pattern.

    **Blob URL hook (src/hooks/useBlobUrl.ts):**
    Create `useBlobUrl(blob: Blob | undefined): string | undefined` that creates object URLs and revokes them on cleanup via useEffect. This prevents memory leaks with 300+ photo thumbnails.

    Add mocks for `createImageBitmap` and `OffscreenCanvas` to test setup since jsdom does not support these APIs.
