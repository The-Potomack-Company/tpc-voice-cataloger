       ```typescript

         throw new Error("VITE_GEMINI_PROXY_URL is not configured. Create a .env file from .env.example.");

       ```
       Move the existing `const proxyUrl = import.meta.env.VITE_GEMINI_PROXY_URL;` (currently line 93) up into this guard block. Remove the duplicate.

    2. **Non-200 response check (add after the fetch call, before parsing candidates):**
       After `const response = await fetch(proxyUrl, {...})`, add:
       ```typescript
       if (!response.ok) {
         const errorText = await response.text().catch(() => "unknown");

       }
       ```
       This ensures 4xx/5xx responses throw before attempting to parse `data.candidates[0]...`.

           });

           console.error("Failed to update aiStatus to failed:", dbError);

       }
